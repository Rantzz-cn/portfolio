## Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- A code editor (VS Code recommended)

### Installation & Setup

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Or use a local server:
   - Using Python 3: `python -m http.server 8000`
   - Using Node.js: `npx http-server`
   - Using VS Code: Install "Live Server" extension and right-click `index.html` â†’ "Open with Live Server"

## Customization

### Update Personal Information
1. Open `index.html`
2. Replace "Your Name" with your actual name
3. Update email in the contact section
4. Add your social media links

### Add Projects
1. Add images to `assets/images/`
2. Update project cards in the Projects section
3. Add project links and descriptions

### Customize Colors
1. Open `assets/css/style.css`
2. Modify CSS variables in `:root`:
   - `--primary-color`: Main color
   - `--secondary-color`: Secondary color
   - `--text-color`: Text color

### Add New Sections
1. Create new `<section>` in `index.html`
2. Add corresponding styles in `style.css`
3. Add navigation link in the navbar

## Deployment Options

### GitHub Pages (Free)
1. Create a GitHub repository
2. Push your portfolio files
3. Enable GitHub Pages in repository settings
4. Your site will be live at `username.github.io`

### Netlify (Free)
1. Connect your GitHub repository
2. Deploy directly from your code
3. Get a free domain or custom domain

### Vercel (Free)
1. Similar to Netlify
2. Great performance and ease of use

### Other Options
- Heroku, AWS, HostGator, Bluehost, etc.
